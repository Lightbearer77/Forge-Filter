# Forge Filter

Longhouse Tribal Forge — Male Candidate Vetting System

## Tabs
- **Score Candidate** — Add candidates, work through Stages I–IV, log field notes, trigger disqualifiers
- **Pipeline** — Overview of all candidates by current stage
- **Approach Guide** — Venue sourcing strategy and approach rules

## Run Locally
```bash
npm install
npm run dev
```

## Deploy to GitHub Pages
```bash
npm run build
```
Push the `dist/` folder to a `gh-pages` branch, or connect Vercel for automatic deploys.
